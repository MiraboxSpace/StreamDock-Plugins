# How to use?

## 1. Get Client ID

Click this link to jump to the discord dev

<img src="./docs/3_1.png">

Create an app and add a redirect address: http://127.0.0.1:26432

<img src="./docs/4_1.png">

## 2. Authorization

After copying the ID, fill it into the discord plugin and click Request Authorization to authorize it before use.
(In version 25.11.04 and subsequent versions, you only need to fill in your ID and secret, click the save button, and no longer really need authorization)
